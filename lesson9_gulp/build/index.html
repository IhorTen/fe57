HELLO GULP

  • 1) NPM - пакетный менеджер устанавливающийся вместе с NodeJS
  • 2) Bower - пакетный менеджер для front-end зависимостей
  • 3) GULP - сборщик front-end приложений

Пакетный менеджер - это набор программного обеспечения позволяющего управлять(скачивать,удалять,обновлять) зависимостями в вашем проекте

Сборщик - выполняет всю рутинную работу за вас. При помощи различных плагинов может выполнять операции по минификации,конкатенации файлов. Компилировать препроцессорные языки в стандартныые, которые умеет читать браузер. Ставить префиксы в CSS, минифицировать картинки и просто переносить, переименовывть файлы Вещь НЕОБХОДИМА в современном мире разработки
